How to join Civilisations Classic guide

Joining our server is easy. All you need is a few mods! Below is a list of mods needed.

Required Mods List

Falling Tree
1.20.1-4.3.4
Trees break completely upon using axe
Collective
1.20.1-8.1
Library for mods
Ore Harvester
[Fabric/Forge/Neo] 1.20.1-1.5 | Ore Harvester
You can mine entire ore vein at once while sneaking. It might take longer and use more durability, but you save some time to move.
Traveler's Backpack
1.20.1-9.1.34
Adds backpacks to the game. You can either craft them or find mobs carrying them.
[TaCZ]Timeless and Classics Zero
1.20.1-1.1.4
Realistic guns with smooth animations. Some guns are limited on server.
Create
1.20.1-6.0.6
Industrial mod that allows you to automate farms, mining and more! Its all in your hands!
The Twilight Forest
1.20.1-4.3.2508-universal
New adventure starts in a whole new dimension! Who knows whats lurking inside.
Infinite Trading
[Fabric/Forge/Neo] 1.20.1-4.6
Infinite trading with villagers
GeckoLib
geckolib-forge-1.20.1-4.8.2
Library for mods
Curios API
curios-forge-5.14.1+1.20.1
Library for mods
Create Crafts & Additions
createaddition-forge-1.20.1-1.3.2

superbwarfare-1.20.1-0.8.7.1-6b9d8bd8a
